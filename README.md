Project Title: Bash Scripting Suite for System Maintenance
Student: Siddharth Raj

reg no: 2241018037
Objective: Automate Linux maintenance tasks (backup, update, log monitoring)
Scripts:
  - backup.sh: Creates a compressed backup of /home directory.
  - update_cleanup.sh: Updates and cleans system packages.
  - log_monitor.sh: Monitors system logs for errors.
  - maintenance_suite.sh: Menu-based controller for all scripts.
  - setup.sh: Prepares folders and permissions.

Execution Steps:
1. Run setup.sh
2. Run maintenance_suite.sh
3. Choose tasks from the menu
4. Check logs in /var/log/maintenance.log and /var/log/alert.log
